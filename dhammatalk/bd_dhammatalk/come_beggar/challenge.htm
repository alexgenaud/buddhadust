Why not?
Because the answer is not here,
That's why not.

Is this how you respond to challenges?
You should be ashamed!
Buck up!
Dig in!
It can be done!

"It can be done, beggars!
It can be done!
If it could not be done,
I would not say:
'It can be done!'
It can be done,
So I say:
'It can be done!'"
