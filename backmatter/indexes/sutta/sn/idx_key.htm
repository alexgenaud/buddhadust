Please note:

The structure of this Nikaya is:

Vagga (Volume, Part, Book, Division)

Samyutta (Book, Section, Chapter)

Vagga (Chapter, Collection, Sub-section)

Sutta (2000+)

In order that this Index not be impossibly long to load, it has been broken up into a structure reflecting the structure of the Nikaya. It will go like this:

First Level: (e.g: Index to the Samyutta Nikaya): Index of Volumes (Vaggas). Clicking on the Name of the Volume will bring you to the Book (Samyutta) Index for that Volume. Also on the first level, under each Volume is a list of the Book indexes; if you know the number or Pali name of the Samyutta you are looking for you can go directly to it's index from this page.

Second Level (e.g: Sagathavagga): Index of Books (Samyuttas); One page for each of the above Volumes, with PTS, WP names and page references and any other Book-level information that comes up. Clicking on the Name of a Book will bring you to the Sutta Index for that Book. (There is no Chapter (Vagga) Level Index under the Book level because most of these sub-divisions are unnamed).

Third Level (e.g. Devatasamyutta): Index of Suttas. One page for each of the above Books. These Indexes contain the full list of Suttas.
