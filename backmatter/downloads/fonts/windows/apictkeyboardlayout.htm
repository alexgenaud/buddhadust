If you do not see characters with American Pictogram in the Left Hand Columns, then you do not have the American Pictogram Font installed. Download this font and install it by unziping the font to the desktop, going to My Computer>Control Panel>Fonts>File>Add New Font, browsing to the font and clicking Install, OK.
