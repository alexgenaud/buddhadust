Aŋguttara Nikāya
Ekā-Dasaka Nipāta

Suttas 973-1042

Adapted from the 1995 edition of the digital version of the Sri Lanka Buddha Jayanti Tripitaka Series.

 

 

[361]

1. [pts] Dosassa ... pe ... mohassa ... pe ... ko'dhassa ... pe ... upanāhassa ... pe ... makkhassa ... pe ... pepalāsassa ... pe ... issāya ... pe ... macchariyassa ... pe ... māyāya ... pe ... sāṭheyyassa ... pe ... thambhassa ... pe ... pesārambhassa ... pe ... mānassa ... pe ... ati-mānassa ... pe ... madassa ... pe ... pepamāssa abhiññāya ... pe ... pariññāya ... pe ... parikkhayāya ... pe ... pepahānānāya ... pe ... khayāya ... pe ... vayāya -pevirāgāya ... pe ... penirodhāya ... pe ... pecāgāya ... pe ... paṭinissaggāya ime ekādasa-dhammā bhāvetabbāti.|| ||

Idam avoca Bhagavā.|| ||

Attamanā te bhikkhu Bhagavato bhāsitaɱ abhinandunti.|| ||

Rāgādipeyyālo niṭṭhito.|| ||

Nava suttasahassāni bhīyo pañca-satāni ca,

Sattapaññāsasuttāni
