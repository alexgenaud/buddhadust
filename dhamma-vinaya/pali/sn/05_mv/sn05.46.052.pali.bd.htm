Saɱyutta Nikāya
5. Mahā-Vagga
46. Bojjhanga Saɱyutta
6. Bojj'Anga-Sākacca Vagga

Sutta 52

Pariyāya Suttaɱ

Adapted from the 1995 edition of the digital version of the Sri Lanka Buddha Jayanti Tripitaka Series.

 

[108]

[1][pts][bodh][than] Evaɱ me sutaɱ:|| ||

Ekaɱ samayaɱ Bhagavā Sāvatthiyaɱ viharati.|| ||

2. Atha kho sambahulā bhikkhū pubbaṇha-samayaɱ nivāsetvā patta-cīvaram ādāya||
Sāvatthiɱ piṇḍāya pavisiɱsu.|| ||

3. Atha kho tesaŋ bhikkhūnaɱ etad ahosi:|| ||

"Atippago kho tāva Sāvatthiyaɱ piṇḍāya carituɱ.|| ||

Yaɱ nūna mayaɱ yena añña-titthiyānaɱ ārāmo paribabājakānaɱ ten'upasankameyyāmā" ti.|| ||

4. Atha kho te bhikkhū yena añña-titthayānaɱ paribbājakānaɱ ārāmo ten'upasankamiɱsu.|| ||

Upasaŋkamitvā tehi añña-titthiyehi paribbājakehi saddhiɱ sammodiɱsu.|| ||

Sammodanīyaɱ kathaɱ sārāṇīyaɱ vīti-sāretvā eka-m-antaɱ nisīdiɱsu.|| ||

Eka-m-antaɱ nisinne kho te bhikkhū te añña-titthiyā paribbājakā etad avocuɱ:|| ||

5. "Samaṇo āvuso Gotamo sāvakānaɱ evaɱ dhammaɱ deseti:|| ||

'Etha tumhe bhikkhave,||
pañca nīvaraṇe pahāya cetaso upakkilese paññāya dubbalī-karaṇe satta bojjh'aŋge yathā-bhūtaɱ bhāvethā' ti.|| ||

Mayam pi kho āvuso,||
sāvakānaɱ evaɱ dhammaɱ desema:|| ||

'Etha tumhe āvuso,||
pañca nīvaraṇe pahāya cetaso upakkilese paññāya dubbalī-karaṇe satta bojjh'aŋge yathā-bhūtaɱ bhāvethā' ti.|| ||

Idha no āvuso ko viseso,||
ko adhippāyo.|| ||

Kiɱ nānā-karaṇaɱ,||
samaṇassa vā Gotamassa amhākaɱ vā||
yad idaɱ Dhamma-desanāya vā||
Dhamma-desanaɱ anusāsaniyā vā anusāsanin" ti?|| ||

6. Atha kho te bhikkhū tesam añña-titthiyānaɱ paribbājakānaɱ bhāsitaɱ n'eva abhinandiɱsu,||
na p-paṭikkosiɱsu.|| ||

Anabhinan'ditvā a-p-paṭikkositvā uṭṭhāy'āsanā pakkamiɱsu:|| ||

"Bhagavato santike etassa bhāsitassa atthaɱ ājānissāmā" ti.|| ||

 

§

 

7. Atha kho te bhikkhū Sāvatthiyaɱ piṇḍāya caritvā pacchābhantaɱ piṇḍa-pāta-paṭikkantā yena Bhagavā ten'upasankamiɱsu.|| ||

Upasaŋkamitvā Bhagavantaɱ abhivādetvā eka-m-antaɱ nisīdiɱsu.|| ||

[109] 8. Eka-m-antaɱ nisinnā kho te bhikkhū Bhagavantaɱ etad avocuɱ:|| ||

"Idha mayaɱ bhante pubbaṇha-samayaɱ nivāsetvā patta-cīvaram ādāya Sāvatthiɱ piṇḍāya pavisimha.|| ||

Tesaŋ no bhante, amhākaɱ etad ahosi:|| ||

'Atippago kho tāva Sāvatthiyaɱ piṇḍāya carituɱ.|| ||

Yaɱ nūna mayaɱ yena añña-titthiyānaɱ paribbājakānaɱ ārāmo ten'upasankameyyāmā' ti.|| ||

Atha kho mayaɱ bhante,||
yena añña-titthiyānaɱ paribbājakānaɱ ārāmo ten'upasankamimha.|| ||

Upasaŋkamitvā tehi añña-titthiyehi paribbājakehi saddhiɱ sammodimha.|| ||

Sammodanīyaɱ kathaɱ sārāṇīyaɱ vīti-sāretvā eka-m-antaɱ nisīdimha.|| ||

Eka-m-antaɱ nisinne kho amhe bhante,||
te añña-titthiyā paribbājakā etad avocuɱ:|| ||

9. 'Samaṇo āvuso Gotamo sāvakānaɱ evaɱ dhammaɱ deseti:|| ||

"Etha tumhe bhikkhave, pañca nīvaraṇe pahāya cetaso upakkilese paññāya dubbalī-karaṇe satta bojjh'aŋge yathā-bhūtaɱ bhāvethā" ti.|| ||

Mayam pi kho āvuso, sāvakānaɱ evaɱ dhammaɱ desema:|| ||

"Etha tumhe āvuso, pañca nīvaraṇe pahāya cetaso upakkilese paññāya dubbalī-karaṇe satta bojjh'aŋge yathā-bhūtaɱ bhāvethā" ti.|| ||

Idha no āvuso ko viseso,||
ko adhippāyo.|| ||

Kiɱ nānā-karaṇaɱ,||
samaṇassa vā Gotamassa amhākaɱ vā||
yad idaɱ Dhamma-desanāya vā||
Dhamma-desanaɱ anusāsaniyā vā anusāsanin' ti?|| ||

10. Atha kho mayaɱ bhante,||
tesam añña-titthiyānaɱ paribbājakānaɱ bhāsitaɱ n'eva abhinandimha,||
na p-paṭikkosimha.|| ||

Anabhinan'ditvā a-p-paṭikkositvā uṭṭhāy'āsanā pakkamimha:|| ||

'Bhagavato santike etassa bhāsitassa atthaɱ ājānissāmā'" ti.|| ||

 

§

 

11. "Evaɱ vādino bhikkhave, añña-titthiyā paribbājakā evam assu vacanīyā.|| ||

'Atthi pan'āvuso pariyāyo||
yaɱ pariyāyaɱ āgamma||
pañca nīvaraṇā dasa honti||
satta bojjh'aŋgā catuddasā' ti?|| ||

Evaɱ puṭṭhā bhikkhave,||
añña-titthiyā paribbājakā na c'eva sampāyissanti||
uttariñ ca vighātaɱ āpajji-s-santi.|| ||

12. Taɱ kissa hetu?|| ||

Yathā taɱ bhikkhave, avisayasmiɱ.|| ||

Nāhaɱ taɱ bhikkhave,||
passāmi sa-devake loke sa-Mārake||
sa-brahmake||
sa-s-samaṇa-brāhmaṇiyā||
pajāya sadeva-manussāya||
yo imesam pañhānaɱ veyyā-karaṇena cittaɱ ārādheyya aññatra Tathāgatena vā||
Tathāgata-sāvakena vā||
ito vā pana sutvā.|| ||

 

Pañca Dassa Honti

 

[110] 13. Katamo ca bhikkhave, pariyāyo yaɱ pariyāyaɱ āgamma pañca nīvaraṇā dasa honti?|| ||

14. Yad api bhikkhave, ajjhattaɱ kāma-c-chando||
tad api nīvaraṇaɱ.|| ||

Yad api bahiddhā kāma-c-chando||
tad api nīvaraṇaɱ.|| ||

'Kāma-c-chanda nīvaraṇaɱ' ti||
iti h'idaɱ uddesaɱ āga-c-chati.|| ||

Tad aminā p'etaɱ pariyāyena dvayaɱ hoti.|| ||

■

15. Yad api bhikkhave, ajjhattaɱ vyāpādo||
tad api nīvaraṇaɱ.|| ||

Yad api bahiddhā vyāpādo||
tad api nīvaraṇaɱ.|| ||

'Vyāpāda nīvaraṇan' ti||
iti h'idaɱ uddesaɱ āga-c-chati.|| ||

Tad aminā p'etaɱ pariyāyena dvayaɱ hoti.|| ||

■

16. Yad api bhikkhave, thīnaɱ||
tad api nīvaraṇaɱ.|| ||

Yad api middhaɱ||
tad api nīvaraṇāɱ.|| ||

'Thīna-middha nīvaraṇaɱ' ti||
iti h'idaɱ uddesaɱ āga-c-chati.|| ||

Tad aminā p'etaɱ pariyāyena dvayaɱ hoti.|| ||

■

17. Yad api bhikkhave, uddhaccaɱ||
tad api nīvaraṇaɱ.|| ||

Yad api kukkuccaɱ||
tad api nīvaraṇaɱ.|| ||

'Uddhacca-kukkucca nīvaraṇan' ti||
iti h'idaɱ uddesaɱ āga-c-chati.|| ||

Tad aminā p'etaɱ pariyāyena dvayaɱ hoti.|| ||

■

18. Yad api bhikkhave, ajjhattaɱ dhammesu vicikicchā||
tad api nīvaraṇaɱ.|| ||

Yad api bahiddhā dhammesu vicikicchā||
tad api nīvaraṇaɱ.|| ||

'Vicikicchā nīvaraṇan' ti||
iti h'idaɱ uddesaɱ gacchati.|| ||

Tad aminā p'etaɱ pariyāyena dvayaɱ hoti.|| ||

19. Ayaɱ kho bhikkhave, parāyāyo||
yaɱ pariyāyaɱ||
āgamma pañca nīvaraṇā dasa honti.|| ||

 

Satta Catuddasa Honti

 

20. Katamo ca bhikkhave, pariyāyo||
yaɱ pariyāyaɱ||
āgamma satta bojjh'aŋgā catuddasa honti?|| ||

21. Yad api bhikkhave, ajjhattaɱ dhammesu sati||
tad api sati-sambojjh'aŋgo.|| ||

Yad api bahiddhā dhammesu sati tad api sati-sambojjh'aŋgo.|| ||

'Sati-sambojjhago' ti||
iti h'idaɱ uddesaɱ āga-c-chati.|| ||

Tad aminā p'etaɱ pariyāyena dvayaɱ hoti.|| ||

■

[111] 22. Yad api bhikkhave, ajjhattaɱ dhammesu paññāya||
pavicinati||
pavicarati||
parivīmaɱsam||
āpajjati||
tad api dhamma-vicaya-sambojjh'aŋgo.|| ||

Yad api bahiddhā dhammesu paññāya||
pavicanati||
pavicarati||
parivīmaɱsam||
āpajjati||
tad api dhamma-vicaya-sambojjh'aŋgo.|| ||

'Dhamma-vicaya-sambojjhago' ti||
iti h'idaɱ uddesaɱ gacchati.|| ||

Tad aminā p'etaɱ pariyāyena dvayaɱ hoti.|| ||

■

23. Yad api bhikkhave, kāyikaɱ viriyaɱ||
tad api viriya-sambojjh'aŋgo.|| ||

Yad api cetasikaɱ viriyaɱ||
tad api viriya-sambojjh'aŋgā.|| ||

'Viriya-sambojjh'aŋgo' ti||
iti h'idaɱ uddesaɱ āga-c-chati.|| ||

Tad aminā p'etaɱ pariyāyena dvayaɱ hoti.|| ||

■

24. Yad api bhikkhave, sa-vitakka-sa-vicārā pīti||
tad api pīti-sambojjh'aŋgo.|| ||

Yad api avitakka-avicārā pīti||
tad api pīti-sambojjh'aŋgo.|| ||

'Pīti-sambojjh'aŋgo' ti||
iti h'idaɱ uddesaɱ āga-c-chati.|| ||

Tad aminā p'etaɱ pariyāyena dvayaɱ hoti.|| ||

■

25. Yad api bhikkhave, kāya-passaddhi||
tad api passaddhi-sambojjh'aŋgo.|| ||

Yad api citta-passaddhi,||
tad api passaddhi-sambojjh'aŋgo.|| ||

'Passaddhi-sambojjh'aŋgo' ti||
iti h'idaɱ uddesaɱ āga-c-chati.|| ||

Tad aminā p'etaɱ pariyāyena dvayaɱ hoti.|| ||

 

■

 

26. Yad api bhikkhave, sa-vitakko sa-vicāro samādhi||
tad api samādhi-sambojjh'aŋgo.|| ||

Yad api avitakko avicāro samādhi||
tad api samādhi-sambojjh'aŋgā.|| ||

'Samādhi-sambojjh'aŋgo' ti||
iti h'idaɱ uddesaɱ gacchati.|| ||

Tad aminā p'etaɱ pariyāyena dvayaɱ hoti.|| ||

■

27. Yad api bhikkhave, ajjhattaɱ dhammesu upekhā||
tad api upekhā-sambojjh'aŋgo.|| ||

Yad api bahiddhā dhammesu upekhā||
tad api upekhā-sambojjh'aŋgo.|| ||

'Upekhā-sambojjhago' ti||
iti h'idaɱ uddesaɱ gacchati.|| ||

Tad aminā p'etaɱ pariyāyena dvayaɱ hoti.|| ||

28. Ayaɱ kho bhikkhave, pariyāyo||
yaɱ pariyāya||
āgamma satta-bojjh'aŋgā catuddasā" ti.|| ||
